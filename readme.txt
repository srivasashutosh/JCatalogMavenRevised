This package contains the source code for the JCatalog Project.

Pcakage structure:
/bin: installation instruction and database scripts.
/dist: web application war file (after build).
/docs: java doc.
/lib: library jar files.
/src: java source files.
/web: web source files, images, and configuration files.

For installation instruction, please refer to /bin/installation.txt.
